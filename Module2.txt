-->  Download MinGW (Minimalist GNU for Windows):

	ðŸ‘‰ https://www.mingw-w64.org/
	Or download from SourceForge:
	ðŸ‘‰ https://sourceforge.net/projects/mingw-w64/

	Install MinGW:

	Run the installer.

	Choose architecture: x86_64 (for 64-bit) or i686 (for 32-bit).

	Choose threads: posix, exception: seh (recommended for 64-bit).

	Set installation directory (e.g., C:\mingw-w64).

	Add GCC to System Path:

	Open Environment Variables from Control Panel.

	Edit the Path variable and add:
	Example:
	C:\mingw-w64\bin

	This allows you to use GCC commands like gcc from anywhere in the Command Prompt.

	Verify Installation:

	Open Command Prompt and type:

	gcc --version

	If installed correctly, it will show GCC version info
